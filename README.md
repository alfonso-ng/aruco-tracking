## Descripción general

Este proyecto presenta un prototipo de software para planificación quirúrgica asistida por ordenador, que combina visión por computador y visualización 3D interactiva.

El sistema permite posicionar y orientar un modelo anatómico 3D en función de la detección de un marcador ArUco, simulando un flujo básico de navegación o guiado quirúrgico.

El objetivo del proyecto es explorar la integración de Unity y OpenCV en un contexto médico, con especial foco en precisión espacial, visualización y prototipado rápido.
